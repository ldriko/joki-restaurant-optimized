# Restowan

Destinasi rekomendasi restoran anda!

## Project setup

```
npm install
```

## Run dev server

```
npm run start-dev
```

## Run unit & integration tests

Jalankan dev server terlebih dahulu untuk memulai
end-to-end testco

```
npm run test
npm run test:e2e
```

## Build for production

```
npm run build
```
